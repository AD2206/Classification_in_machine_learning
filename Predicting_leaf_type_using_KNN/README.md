This machine learning model is used the predict the leaf type .The factors that are taken into consideration are petal length,petal width,sepal length and sepal width.All these
dimensions are inputted by the user.The leaves that we have taken into consideration are 'setosa','virginica','versicolor'.Our machine learning model uses the method of
K-Nearest Neighbors.By standard and best practice,n_neighbors have taken as 5 and minkowski method has been applied with euclidian distance being used to measure the distance.
I happy to claim that our model has an accuracy of 97.368% and our confusion matrix has only 1 displaced element.
